# Avatars

이 폴더에 Ready Player Me에서 다운로드한 GLB 파일을 저장하세요.

## 다운로드 방법

1. https://readyplayer.me 접속, 회원가입
2. "Create avatar" → 아바타 생성
3. 완성 후 URL 복사 (형식: `https://models.readyplayer.me/[ID].glb`)
4. URL을 브라우저에 붙여넣어 .glb 파일 다운로드
5. 이 폴더에 `male.glb`, `female.glb` 같은 이름으로 저장

## 권장 설정

- 옷 없는 (또는 단순한 흰색 티셔츠) 아바타 추천
- AI가 옷을 그려야 하므로 깔끔한 베이스 모델이 좋음
